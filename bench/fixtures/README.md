# Benchmark Fixtures

Provenance for all files in this directory.

## Text fixtures

| File | Size | Description |
|---|---|---|
| `js/small.js`        | ~4 KB  | Hand-written utility module (string/number/array/DOM helpers) |
| `js/medium.js`       | ~16 KB | Larger utility library (same categories + colour, store, async, file utils) |
| `css/small.css`      | ~3 KB  | Base component stylesheet (reset, typography, buttons, form inputs) |
| `css/medium.css`     | ~14 KB | Full component library (custom props, layout, navbar, cards, forms, table, dark mode) |
| `css/for-sass.css`   | ~4 KB  | Multi-rule CSS with nested selectors — designed to exercise the CSS→SASS converter |
| `html/small.html`    | ~4 KB  | Simple tool page template |
| `html/medium.html`   | ~18 KB | Full home page with hero, tool grid, FAQ, footer |
| `scss/medium.scss`   | ~7 KB  | SCSS with variables, maps, mixins, nesting, BEM components |

## Binary fixtures (used by browser benchmark only)

Binary fixtures are served from `public/bench-fixtures/` by the Astro dev server.

| File | Size | Description | How to generate |
|---|---|---|---|
| `fonts/sample.ttf`   | ~401 KB | Liberation Sans Regular | Copied from `/usr/share/fonts/truetype/liberation/LiberationSans-Regular.ttf` |
| `fonts/sample.woff2` | ~160 KB | WOFF2 version of sample.ttf | Generated by `bench/node/bench-font-converter.ts` beforeAll or manually via wawoff2 |

See `bench/node/bench-font-converter.ts` for how to regenerate `sample.woff2` if needed.
